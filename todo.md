# PMP System - 待辦清單 (To-Do List)

本文件用於追蹤系統後續可優化項目與功能迭代。

## 🚀 即將進行 / 建議優化

### 🏢 核心模組與權限
- [ ] **精細化 Menu 權限**：目前側欄雖然已過濾路由，但部分操作按鈕可過濾得更精準（例如 PM 只能編輯自己專案的資訊）。
- [ ] **審核記錄歷史 Log**：WBS 版本與變更請求（CR）同意／拒絕時，補上操作紀錄者的顯示與時間戳記 Log 集合。
- [ ] **多主管部門權限**：支援一個 Manager 可以管理多個部門（目前 Mongoose Schema 只綁定單一 `department: string`）。

### 🔌 系統整合 (Integrations)
- [ ] **補齊 Entra ID 同步邏輯**：目前 `integrations.ts` 中對 Entra ID 的同步主要為骨架，需整合 Graph API 完成背景人員自動排程同步。
- [ ] **修補電子郵件／通知整合**：整合 SMTP 或 Azure Communication Services，可在「待審核」或「毛利警告」時自動發送 Email 警報給負責人。
- [ ] **Teams / Slack 通知整合**：除了 Email，整合企業常用的通訊軟體，當有「待審核 WBS」或「工時未填提醒」時，直接發送到 Teams 頻道（或透過 Bot 私訊）。

### 📊 分析與結算 (Analytics)
- [ ] **利潤中心制 KPI 與結算**：將現有 KPI 與結算邏輯改為「利潤中心制公式」，確保專案與各部門的利潤貢獻被精確量化與分配。
- [ ] **人力資源熱點圖 (Resource Heatmap)**：在資源池中加入視覺化的「負載分析」，一眼看出哪些工程師工作量過載（Over-utilized），哪些人還有餘裕，輔助 PM 派工調度。
- [ ] **自訂報表產生器 (Report Builder)**：允許管理者自行選擇維度（日期、部門、專案性質）並選取欄位，組合出自己需要的 Excel 或介面報表。
- [ ] **KPI 自訂圖表時間軸**：KPI 看板目前為寫死的統計，可加入「月份／季度區間篩選」。
- [ ] **匯出 PDF 功能**：除了 CSV 匯出，月度結算或 KPI 視圖可加上 PDF 匯出格式以供列印需求。

### 🎨 視覺與體驗 (UI/UX)
- [ ] **商務質感 (Business Premium) 視覺全面升級**：
  - 更新配色系統 (OKLCH: 深藍/石板灰) 以營造專業洗練感。
  - 導入精緻的陰影與邊框層次設計，提升介面深度。
  - 優化微互動動畫與現代化字體 (Inter/Outfit) 導入。
  - 確保所有模組在「商務質感」風格下保持視覺一致性。
- [ ] **全域快速搜尋 (Command+K)**：模仿現代化系統，提供一個浮動搜尋框，可以快速跳轉到任何「專案」、「商機」或「工程師個人頁面」。
- [ ] **介面多語系切換**：支援繁中、英文切換，因應未來可能的跨國商機或多國團隊協作。

- [ ] **協銷紀錄附件上傳與版本控管**：

  - **檔案版本管理 (Versioning)**：對上傳至 SharePoint 的文件進行版本控制，可追蹤 SOW 或報價單的修訂歷史。
  - 規劃 SharePoint 實體儲存架構與未來「商機轉專案」時的自動化建立機制。
- [ ] **PMP 模組功能強化**：
  - [ ] 新增議題管理（Issue Management）功能。
  - [ ] WBS 補強進度百分比、顏色、完成度顯示。
  - [ ] **WBS 優化與匯出**：支援匯出成 `.xlsx` 格式，欄位包含：`WBS Code`、`工作名稱`、`層級`、`工作說明`、`主要負責人`、`估計工期(天)`、`前置任務`、`備註`。
